# Media — Images & Videos for the Website

Drop your images and videos here. When you're ready to use them on the site, tell Claude Code and I'll move them to the right place in the site assets folder and update the HTML.

## Folders

```
media/
  images/    <-- Blog thumbnails, hero images, section graphics
  videos/    <-- Short clips, explainer videos, testimonials
```

## Naming Convention

Name files by what they're for so it's obvious later:

**Images:**
- `blog-stop-losing-leads-thumb.png` — thumbnail for a specific article
- `hero-lab-atmosphere.png` — hero section background
- `section-trust-stats.png` — section graphic
- `testimonial-mark-reeves.jpg` — client photo

**Videos:**
- `explainer-how-it-works.mp4` — homepage explainer video
- `testimonial-mark-reeves.mp4` — client video testimonial
- `blog-zoho-crm-demo.mp4` — screen recording for an article

## Size Guidelines

- **Images:** Compress with Squoosh (squoosh.app) to under 200KB each before using on the site
- **Videos under 30 seconds:** Fine to store here and in the site repo (keep under 10MB each)
- **Videos over 30 seconds:** Upload to YouTube and embed on the page instead — faster loading and the YouTube presence helps AI search rankings
